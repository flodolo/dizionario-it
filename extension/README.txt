Version 5.1, 13/12/2020 (DD/MM/YYYY)

Starting with version 5.1, this is an independent fork of "Estensione
linguistica italiana - Italian Writing Aids extension" by
https://libreitalia.org version 5.1.0

You can find the README for the original package in README_LIBRE.txt.

It's designed to be used in Mozilla products, like Firefox and Thunderbird,
and can be installed from
https://addons.mozilla.org/firefox/addon/dizionario-italiano/

disponibile nel file GPLv3.txt.

ENGLISH

The "Dizionario italiano" is an add-on for Mozilla products, like Firefox and
Thunderbird, and provides support for Italian language in the built-in
spellchecker feature.

The current maintainer of this packages is Francesco Lodolo
(flod<AT>lodolo<DOT>net). The add-on is released under the terms and conditions
of the GNU General Public License (GPL) 3, a copy of which is available in the
GPLv3.txt file.

The Openoffice.org Community, PLIO http://www.plio.it and LibreItalia
https://libreitalia.org contributed to the previous versions of the Italian
dictionary used as foundation for this add-on. Read README_LIBRE.txt for more
information.
